Удаление репозитория git
rm -fr .git

Игнорирование файлов
git rm --cached <file_name> Делает файл неотслеживаемым для ГИТа

Настройка, чтобы команда git add не добавляла hint'ы для заигноренных файлов
git config advice.addIgnoredFile false, однако команда git add -f <file_name> все равно позволяет индексировать изменения в заигноренных файлах
